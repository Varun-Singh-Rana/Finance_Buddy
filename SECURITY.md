# Security Policy

## Supported Versions
Currently, only the latest version of FinLytics is actively maintained and supported.

## Reporting a Vulnerability
If you discover a security vulnerability, please do NOT open a public issue.

Instead, report it privately via:
- Email: syncforgeteam@gmail.com

We will review the issue and respond as soon as possible.

## Security Measures
This project includes:

- Local encrypted SQLite database
- No cloud data transmission
- Input validation to prevent SQL injection
- Offline-first architecture for data privacy

## Responsible Disclosure
We appreciate responsible disclosure and will credit contributors who report valid security issues.
